# Ansible Role: PostgreSQL Server for Debian 12

[![Ansible Role](https://shields.io)](https://ansible.com)
[![Testing Framework](https://shields.io)](https://readthedocs.io)
[![Platform](https://shields.io)](https://debian.org)
[![Provider](https://shields.io)](https://libvirt.org)

Ansible-роль для автоматической чистой установки, настройки и инициализации сервера баз данных **PostgreSQL 15** на операционной системе **Debian 12 (Bookworm)**. 

Роль полностью протестирована локально с помощью фреймворка **Molecule** внутри инфраструктуры виртуализации **Vagrant** под управлением **libvirt/KVM**.

---

## 🚀 Возможности роли

* Автоматическое обновление кэша пакетов APT.
* Установка СУБД `PostgreSQL` и необходимых расширений (`contrib`).
* Установка системных утилит доступа `acl` для безопасного переключения привилегий (`become_user`).
* Установка библиотек `python3-psycopg2` для нативного управления базами данных через Ansible.
* Управление системной службой (добавление в автозагрузку и запуск через `systemd`).
* Автоматическое создание кастомного пользователя СУБД с безопасными флагами прав.
* Автоматическое создание целевой базы данных и назначение её владельца.

## 🧪 Тестирование роли с помощью Molecule (Локально)

Окружение настроено для локального тестирования роли непосредственно внутри виртуальной машины **Vagrant (провайдер libvirt/KVM)** без использования Docker, что исключает конфликты пространства имен ядра (`cgroups v2`) и umask.

### Требования на хост-машине тестирования:
* Наличие установленного `Molecule` и пакета `ansible-core`.

### Файлы конфигурации тестов:
Сценарий инициализирован по пути `molecule/default/` и использует делегированный драйвер (`driver: default`) для выполнения тестов на локальной системе.

* **`molecule/default/molecule.yml`**: Настроен на работу с локальным транспортом.
* **`molecule/default/converge.yml`**: Передает тестовые переменные окружения и запускает роль локально (`connection: local`).

### Запуск тестов:

1. Перейдите в каталог роли:
   ```bash
   cd roles/postgresql
   ```
2. Выполните цикл наката конфигурации и тестирования роли:
   ```bash
   molecule converge
   ```
